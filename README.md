# HYPR Compplan

Sistema de calculo de bonus do time de Customer Success (Compplan 2026).

## Estrutura

- backend/  - Cloud Functions Gen2 (Node 20 + Express)
- frontend/ - React 18 + Vite + Vercel

## Stack

- Backend: Node 20, Express, BigQuery
- Frontend: React 18, Vite, recharts
- Auth: JWT HS256 + Google OAuth (@hypr.mobi)
- Region: us-central1
